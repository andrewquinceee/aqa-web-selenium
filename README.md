# Домашнее задание: Тестирование веб-интерфейсов

## Описание
Автотесты для формы заказа карты на Selenium.

## Запуск локально

1. Запустите тестируемое приложение:
   java -jar ./artifacts/app-order.jar

2. Запустите тесты:
   gradle test

## Технологии
- Java 17
- Selenium WebDriver
- JUnit 5
- Gradle
- GitHub Actions
